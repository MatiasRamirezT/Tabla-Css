¿Qué es CSS?

CSS son las siglas en inglés de Cascading Style Sheets, que significa “hojas de esilo en cascada”. Es un lenguaje que se usa para estilizar elementos escritos en un lenguaje de marcado como HTML. CSS fue desarrollado por W3C (World Wide Web Consortium) en 1996 por una razón muy sencilla. HTML no fue diseñado para tener etiquetas que ayuden a formatear la página. Está hecho solo para escribir el marcado para el sitio. Se incluyeron etiquetas como <%font%> en HTML versión 3.2, y esto les causó muchos problemas a los desarrolladores. Dado que los sitios web tenían diferentes fuentes, fondos de colores y estilos, el proceso de reescribir el código fue largo, doloroso y costoso. Por lo tanto, CSS fue creado por W3C para resolver este problema. La relación entre HTML y CSS es muy fuerte. Dado que HTML es un lenguaje de marcado (es decir, constituye la base de un sitio) y CSS enfatiza el estilo (toda la parte estética de un sitio web), van de la mano. CSS no es técnicamente una necesidad, pero no querrás tener un sitio que solo tenga HTML, ya que se vería completamente desnudo.

¿Por qué es importante JavaScript?

En la creación de experiencias interactivas y dinámicas en la web JavaScript nos permite crear cosas tan distintas como aplicaciones web complejas, juegos en línea o efectos visuales impresionantes. Además, JavaScript es compatible con todos los navegadores web modernos. Según la manera de como lo empleemos, podemos clasificar el uso de JS en 2 grandes contextos: cuando es utilizado para el desarrollo front-end y cuando lo explotamos para desarrollo back-end

Tabla de Atributos CSS

Atributo Descripción Valores Ejemplo Imagen Fuente
font-size Define el tamaño del texto. px, em, %, etc. p { font-size: 16px; } MDN
border-radius Redondea las esquinas de un elemento. px, %, etc. div { border-radius: 10px; } MDN
display Especifica el tipo de caja de un elemento. block, inline, flex, grid, etc. span { display: block; } MDN
flex Define cómo un ítem flexible crecerá o encogerá. none, [número], etc. div { flex: 1; } MDN
position Determina el método de posicionamiento de un elemento. static, relative, absolute, fixed, sticky div { position: absolute; } MDN
box-shadow Agrega sombra alrededor del marco de un elemento. none, [offset-x] [offset-y] [blur-radius] [spread-radius] [color] div { box-shadow: 2px 2px 5px grey; } MDN
overflow Controla qué sucede cuando el contenido se desborda. visible, hidden, scroll, auto div { overflow: hidden; } MDN
z-index Define el orden de apilamiento de elementos. auto, [número] div { z-index: 10; } MDN
opacity Establece la opacidad de un elemento. Valor entre 0.0 (transparente) y 1.0 (opaco) div { opacity: 0.5; } MDN
margin Establece el margen exterior de un elemento. px, em, %, auto, etc. div { margin: 20px; } MDN
grid Define un contenedor de cuadrícula para diseño bidimensional. none, inline, subgrid, etc. div { display: grid; } MDN
transition Define la transición entre valores de propiedades. [propiedad] [duración] [función de temporización] [retraso] div { transition: all 0.3s ease; } MDN
transform Aplica transformaciones 2D o 3D a un elemento. none, translate(), rotate(), scale(), etc. div { transform: rotate(45deg); } MDN
clip-path Define una región de recorte para un elemento. none, circle(), ellipse(), polygon(), url(), etc. div { clip-path: circle(50%); } MDN
